AWS Re:Invent 2025, Reinvented — Powered by MCP

October 27, 2025 · 4 min read
Manu Mishra
Solutions Architect & Applied Software Engineer

Every year, AWS re:Invent brings together thousands of builders, leaders, and innovators to explore the future of cloud. In 2025, the catalog is bigger than ever — 1,843 sessions across 53 areas of interest and 19 industries. Inspiring, yes — but also overwhelming.

That's why I built the re:Invent 2025 MCP Server: a comprehensive Model Context Protocol server that transforms how professionals navigate AWS's flagship conference, providing intelligent access to the complete session catalog with advanced search capabilities and speaker discovery.

🚀 Quick Start for Claude Desktop Users​

Get instant access to all 1,843 re:Invent 2025 sessions in Claude Desktop:

📦 Download Claude Extension​

Installation: Just drag & drop the .mcpb file into Claude Desktop Extensions → Click "Install" → Start exploring!

Try asking Claude: "Find all AI sessions at re:Invent 2025" or "Show me sessions for developers about serverless"

🌟 What It Is​

The re:Invent 2025 MCP Server transforms how professionals navigate AWS's flagship conference. Built on the Model Context Protocol, this server bridges the gap between conference content and actionable insights.

Key Features:

  • 🔍 Universal Search: Query across titles, abstracts, topics, and metadata
  • 👥 Speaker Discovery: Find speakers by name or company with session mapping
  • 🎯 Advanced Filtering: 13 specialized tools for precise content discovery
  • ⚡ Performance Optimized: Sub-second startup with <10ms response times

And here's something important: 🔒 No internet required. No tracking. All requests stay between you and your AI assistant. I don't see or log anything — it's your data, your control.

🎯 Who It's For​

💻 Developers & Architects — Discover hands-on workshops, certification-aligned content, and service-specific sessions that accelerate skill development.

📊 Executives & Leaders — Access strategic frameworks, industry-specific insights, and quantified business outcomes that inform decision-making.

🤝 Sales & Partners — Research customer industries, identify co-marketing opportunities, and extract compelling value propositions from 1,843 sessions.

📅 Event Teams & Analysts — Analyze content gaps, optimize scheduling, track speaker participation across the world's largest cloud computing conference.

✨ Available MCP Tools​

Core Discovery Tools​

  • search_sessions - Universal session search or complete listing
  • search_speakers - Speaker discovery with session mapping
  • get_session_details - Complete session information
  • search_services - AWS service-specific sessions

Specialized Filters​

  • get_sessions_by_level - Foundational → Distinguished (5 levels)
  • get_sessions_by_role - Developer, Architect, Data Scientist, etc. (19 roles)
  • get_sessions_by_industry - Healthcare, Financial Services, Government, etc. (19 industries)
  • get_sessions_by_topic - AI, Security, Databases, Serverless, etc. (18 topics)
  • get_sessions_by_area_of_interest - Generative AI, DevOps, Cost Optimization, etc. (53 areas)

🛠️ How to Get Started​

One-Click Installation (Claude Desktop)​

📦 Download Extension - Just drag & drop into Claude Desktop!

Command Line Installation​

Node.js:

npx reinvent2025mcp

Test with MCP Inspector:

npx @modelcontextprotocol/inspector npx reinvent2025mcp

Integration with AI Assistants​

Amazon Q Developer (CLI):

q mcp add --name reinvent-2025 --command npx --args reinvent2025mcp

Claude Desktop Manual Config:

{
  "mcpServers": {
    "reinvent-2025": {
      "command": "npx",
      "args": ["reinvent2025mcp"]
    }
  }
}
